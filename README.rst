Youtube Remote
===============================================================================

*WIP*


Setup dependencies
----------------------------------------------------------------------

- ``python34+``
- ``virtualenv`` recommended
- ``npm`` for installing
    + ``vue.js``
    + ``semantic ui``

::

    $ make


Run Broker
----------------------------------------------------------------------

::

    $ python3 app.py [host [port]]
